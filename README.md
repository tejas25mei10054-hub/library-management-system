# Library-Management-System
A simple command-line Library Management System in Python to manage books and users efficiently. Features include adding, searching, issuing, returning, and deleting books with user registration. Designed with modular functions and menu-driven interface for easy interaction and basic library operations.
Features
Add new books with details (Title, Author, ISBN)

List all available books

Search books by ISBN

Delete books by ISBN

Register new users

Issue books to registered users

Return books to the library

Display all issued books

User-friendly menu-driven interface

Technologies/Tools Used
Python 3

Basic data structures (lists, dictionaries) for in-memory data management

Command-line interface (CLI)

Installation & Running the Project
Ensure Python 3 is installed on your system.

Download or clone the project files from the GitHub repository.

Open a terminal/command prompt in the project directory.

Run the program with:
python library_management_system.py
Follow the on-screen menu prompts to use the system.

Testing Instructions
Test all menu options by selecting them from the input prompt.

Add books with unique ISBNs and verify listing.

Search existing and non-existing books to observe system responses.

Register users and test issuing books to both registered and unregistered users.

Return issued books and verify the updated status.

Test invalid inputs and menu selections to confirm error handling.
